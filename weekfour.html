This week I learned about how positioning is essential when it comes to building complex interfaces. There are 5 different positioning types static, relative, fixed, absolute, and sticky. I have used a couple of these before learning about them this week, for example I have used the fixed position for the footer of my main blog page so that it will never move from the bottom of the screen. Once the elements position type is set we can use the right, left, top, and bottom properties to place items exactly where we want. This tool makes it easy to layout your webpage exactly how you see fit. My favorite position is the sticky position because once you scroll to where a div with the stick property should be off of the screen it sticks to that part of the screen so that it will always be present. The image at the bottom of this blog post gives a great graphic example of all of the positioning properties so I will keep this in mind when using these properties within my static website.

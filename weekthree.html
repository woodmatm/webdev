This week I learned so much about using flexbox to sort my html code through CSS. Flexbox was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities. I found this article by Mozilla, that along with our book taught me different ways I could organize my webpages and the data within them. The main two CSS elements I learned about are justify-content and align-items. The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container. The CSS align-items property defines the default behavior for how items are laid out along the cross axis. Throughout the week I struggled to get the hang of the concepts but after loads of practice by either playing Flexercise or just doing some simple trial and error, I feel alot better with the progress I have made.

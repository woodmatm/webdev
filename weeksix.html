Wait until multiple promises are complete

I found an article called 10 Clever JavaScript Tricks That Every Developer Should Know. The title really caught my attention so I obviously had to read more. One trick I found that could be very useful in the future was some code that allows you to wait until multiple promises are complete to proceed. It uses the await keyword which was also in our textbook and puts it to good use. Using this Promise.all will also return a rejected if any of the promises return a reject so using this can confirm all or nothing. You can use this trick if you are developing a micro-services architecture and you need to fetch important, non-sequential data from multiple endpoints as fast as possible.

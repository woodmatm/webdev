I really cannot believe how fast this semester has flown by. Professor Johnson, thank you for a great semester and for all your help on the projects when I had errors. I feel like I have learned alot about frontend development in this class which could become very useful in the near future. If I were to redesign this course I would make a few tweaks. I would probably split the course into 2 different course, WebDev I and WebDev II. In WebDev I, I would focus solely on HTML and CSS. I would format the class to have some tiny simple socrative quizzes at the beginning of class to make sure students did the reading. Then on one of the days I would have a code along session where the teacher codes and the student follow along for a lab grade and on the other day I would have them do a similar type lab with a partner with their new aquired knowledge. Instead of doing a simple project I would have it be more like a huge final project with check-ins throughout the semester to clean everything up and then show the final product at the end of the semester. For example, a huge website that could actually be deployed and used by others. I would format WebDev II the exact same way but it would fully teach you javascript and react with the final project being a huge React App that gets deployed. The reason I would format the classes like this is because this semester learning all this new information was a bit rushed and instead I think we could take a deeper dive into each one of these to learn it inside and out. They are very useful tools that will be used in most peoples professional careers at some point in time.

What helped me learn in this class was just doing the labs and paying attention but what I think could help others in the future is by showing more code examples from scratch so we can understand how some of the concepts work within our projects. The book has also been a major help when it comes to learning and working on the projects so thank you for making such an informative book that is easy to follow along with. I really wish there was a second part of this course I could take next semester to expand my knowledge on frontend development so I could pursue a career as a fullstack developer. This has been one of my favorite CS classes at JMU becuase the material is actually very interesting and interactive.
